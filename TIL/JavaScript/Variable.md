# 변수란
변수는 데이터를 담을 수 있는 그릇이다.
데이터들은 컴퓨터의 메모리의 어딘가에 저장되는데 그 값들을 구별하고 재사용하기 위해서 변수를 사용한다.

# 변수 선언하기
변수를 사용하기 위해서는 변수를 선언해야 한다.
```js
var a = "var keyword"; // 재선언, 재할당 가능하며 호이스팅이 발생한다.
let b = "let keyword"; // 재선언 불가능하지만 재할당은 가능하다.
const c = "const keyword"; // 재선언, 재할당 모두 불가능하다. 주로 상수인 변수에 사용한다.
```

# 변수와 대입 연산자
변수에 값을 넣고 싶을 때, 즉 할당할 때에 `=` 연산자를 사용한다.
`=` 연산자는 수학에서의 같다라는 의미가 아니라 변수에 값을 할당한다는 의미의 대입 연산자이다.
자바스크립트에서 `같다`라는 표현은 `===`을 사용한다.

# 변수 사용하기
```js
var a = 7;
var b = 10;
var c = a + b; // 17
a = a + 1; // 8
c = a + b; // 18
```
