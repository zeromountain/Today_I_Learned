# Data-type

## What is Data-type?
- It gives the computer what kind of data was used.
- Each data type has a separate operator that can be used.
- **Operation can be performed only when the data types are the same.**

> Number, String, Boolean, Null, Undefined || Object

### Type Number 
Both integers and real numbers are of type number.

```js
var num = 5;
var positiveInt = 3.0;
var negativeInt = -4.8;
```
#### Operation
```js
+, -, /, (), %
```

### Type String
```js
var str = "I am String";
var adding = "Hello " + "World!"
```
Strings have immutable properties.

#### typeof Operator
```js
var str = "I am String";
typeof str; // Stiring
```
typeof operator shows the type of data.

### Type Boolean
Boolean has both true and false.

#### Comparison operator
```js
5 === "5"; // true
5 < 10 // true
5 > 10 // false
```
#### Logical operator
Logical operators have &&, ||,! Etc.

### Type Null 
null means there is no value.


### Type Undefined
undefined means that the data is not defined.
