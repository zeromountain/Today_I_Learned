## 함수를 만들어 리턴하는 함수 (클로저를 만들어 리턴하는 함수)
- addMaker

```js
  const addMaker = a => b => a + b; // 클로저함수를 리턴하는 함수
  const add10 = addMaker(10);
  console.log(add10); // b => a + b
  console.log(add10(5)); // 15
  console.log(add10(10)); // 20
```
