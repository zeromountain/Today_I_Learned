# 평가
- 코드가 계산(Evaluation)되어 값을 만드는 것
```js
  1 // 평가 => 1
  1 + 2 // 평가 => 3
  (1 + 2) +  4 // 평가 => 7
```

# 일급
- A : 값으로 다룰 수 있다
- B : 변수에 담을 수 있다
- C : 함수의 인자로 사용될 수 있다
- D : 함수의 결과로 사용될 수 있다

```js
  const a = 10; // A, B 
  const add10 = a => a + 10; // C,D
  const r = add10(a); // A, B, C, D
  log(r);
```
